# 📝 TypeScript To-Do List

A simple and responsive To-Do List application built with **HTML**, **CSS**, and **TypeScript**. This project demonstrates DOM manipulation, event handling, local storage, and TypeScript fundamentals.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 💾 Save tasks using Local Storage
* 🔄 Tasks remain after refreshing the page
* 📱 Simple and responsive user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* TypeScript
* JavaScript (compiled from TypeScript)
* Local Storage API
